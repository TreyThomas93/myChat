# myChat
Simple pseudo chat app that allows the user to register a username and chat with other users and view their chat history. Uses sockets to allow for a continuous connection between all users.

## Features
<b>User Allowance: </b>

 - [Chat]
 - [View History]

 - [Login]
 - [Register]
